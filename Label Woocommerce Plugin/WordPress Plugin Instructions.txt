INSTALLING PLUGIN

OPEN YOUR WEB DIRECTORY ON WORDPRESS AND SELECT AND OPEN “wp-content”

THEN SELECT THE “plugins” FOLDER

IN THE “plugins” FOLDER YOU NEED TO CHANGE AND DROP THE “label” directory here into “plugins”

ONCE YOU’VE ADDED THE “label” DIRECTORY TO “plugins” OPEN WORDPRESS

GO TO PLUGINS AND INSTALL THE “Label For Woocommerce” APP
